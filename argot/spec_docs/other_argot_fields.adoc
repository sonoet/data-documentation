= Other Argot fields
Documentation of miscellaneous simpler Argot fields.

toc::[]

== edition

Follows `labeled_value` Argot pattern.

=== UNCb6995648 -- 250 with $3 label
[source]
----
=245  12$aL'equitazione nell'antica Roma /$cAntonio Sestili.
=250  \\$3Vol. 2:$a1a ed.
----

[source,ruby]
----
[
 {'label' => 'Vol. 2', 'value' => '1a ed.'}
]
----
